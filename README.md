
# Cat Dog Detector App Flutter

An app that detects cat and dog and display the result

## Packages Used 

- tflite
- image_picker

# Workings 
- first trained a model of dataset with 8000+ images of Cats and Dog

- create tflite model
- used tflite flutter package to display the result.
